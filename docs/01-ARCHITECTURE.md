# Arquitetura do Projeto

O projeto utiliza React + TypeScript.

A arquitetura deve seguir separação de responsabilidades.

---

# Princípios

O código deve priorizar:

- Componentização.
- Reutilização.
- Organização.
- Manutenção simples.
- Responsividade.

---

# Camadas

## Components

Responsáveis pela interface reutilizável.

---

## Pages

Responsáveis pela composição das telas.

---

## Routes

Responsáveis somente pela navegação.

---

## Services

Responsáveis pela comunicação externa.

---

## Types

Responsáveis pela tipagem dos dados.

---

## Utils

Responsáveis por funções auxiliares independentes.
